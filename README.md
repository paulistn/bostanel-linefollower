# Bostănel - linefollower
Bostănel is an obstacle avoiding line follower robbot created by [ehrmzz](https://github.com/ehrmzz) and me.

This is was Uni project we created and usses a few sensors and actuators like the MH-B IR Sensors or the 1 HC-SR04 Ultrasonic Sensor such that it can follow a line and avoid obstacles. More information about how this robot works can be found in the PDF presentation above. >_<
